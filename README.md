## How to Run

Load the following in terminal:
~~~
$ cd src
$ export DYLD_LIBRARY_PATH=/opt/homebrew/lib:$DYLD_LIBRARY_PATH
$ g++ -o game *.cpp -lsfml-graphics -lsfml-window -lsfml-system
$ ./game
~~~
